# iView-project Template

在iView基础上做的一个模版

Vue.js 2 + vue-router + webpack2 + iView 2 + Vuex

## Install
```bush
// install dependencies
sudo npm install --registry=http://registry.npm.taobao.org
```
## Run
### Development
```bush
// For the first time, run init to create index.html
npm run init
npm run dev
```
### Production(Build)
```bush
npm run build
```
